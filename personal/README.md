# NVIM CONFIG FOR ME
This contains:
- A Completion engine: defaults with Pyright, lua-ls, rust_analyzer, clangd
- Indenting blankline to make scope more 'visible'
- Tokyonight theme
- Telescope
- Lualine

## REQUIREMENTS
- ripgrep
- git
- nvim 0.11+ i think ...
- fd

## SHORTCUTS
- Ctrl-s: open the tree
- a, d, x, r: add, delete, cut, rename file in tree
- Ctrl-e, Ctrl-q: forward, backward on suggestions list
- Ctrl-space: accept suggestion
- space + sa : select all text
- /[regex] + n/N: find [regex] and center the line on the screen
- VISUAL + J/K: move lines selected up/down and indent properly
- Ctrl-t: open terminal or navigate to it
